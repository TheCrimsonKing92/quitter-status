Welcome! This website checks the status of popular Quitter (GNU Social) instances. Currently, only a handful of instances are included (preferably those that support Qvitter), but let me know if you want your instance added here and I'll consider it.

Confused about how GNU Social works? Read the unofficial user manual!

Heads up! The results that you see on this page might not be accurate. Because HTTP requests are made locally, sometimes it might only be your internet that's acting up.
